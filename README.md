# ✨ Nitasha Tarot Website

A visually immersive and elegant portfolio website designed for a tarot reader and spiritual guide, focused on storytelling, branding, and a calming user experience.

🔗 **Live Site:** https://YOUR-NETLIFY-LINK

---

## 📌 Overview

This project was designed and developed as a personal brand website for Nitasha, a tarot reader and spiritual guide.  

The goal was to create a **soothing, intuitive, and visually rich digital experience** that reflects her spiritual identity while guiding users toward learning content, testimonials, and booking sessions.

---

## ✨ Features

- 🌙 Aesthetic, spiritual-themed UI with soft gradients and visuals  
- 🎥 Learn Tarot section with embedded video content  
- 🌍 Testimonials section with global community visualization  
- 📅 Booking section with clear contact and scheduling details  
- 📱 Fully responsive design for mobile and desktop  
- 🎨 Consistent branding aligned with tarot/spiritual identity
